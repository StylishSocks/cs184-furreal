FurReal: Interactive Real-Time Hair and Fur Simulation

Hair and fur are visually important but difficult to simulate in real time. This project builds a modular strand simulation pipeline for dense fur and styled hair, with runtime strategy switching, attachment proxies, and interactive controls.
